## 🎨 UI/UX 改进

### 折叠图标
- **展开状态**：使用 `FluentIcon.CARET_DOWN` 图标
- **折叠状态**：使用 `FluentIcon.CARET_RIGHT` 图标

### 颜色标识
- **严重程度**：红色(严重) > 橙色(高) > 黄色(中) > 蓝色(低)
- **评分**：绿色(优秀≥8) > 橙色(良好≥6) > 红色(及格<6)
- **建议修复**：浅绿色背景 (#e8f5e9)

### 信息展示
- **折叠时**：只显示核心信息（标题、类型、时间、摘要）
- **展开时**：显示完整详情（描述、代码片段、建议等）

## 🔍 技术实现

### 核心机制
```python
class CollapsibleCard(CardWidget):
    def __init__(self, title: str, parent=None):
        super().__init__(parent)
        self._is_expanded = True
        self._setup_ui(title)
    
    def _toggle(self):
        """切换折叠状态"""
        self._is_expanded = not self._is_expanded
        if self._is_expanded:
            self.content_widget.show()
        else:
            self.content_widget.hide()
```

### 批量操作
```python
def _expand_all(self):
    """展开所有卡片"""
    for i in range(self.layout.count()):
        item = self.layout.itemAt(i)
        if item and item.widget():
            widget = item.widget()
            if isinstance(widget, CollapsibleCard):
                widget.set_expanded(True)
```
